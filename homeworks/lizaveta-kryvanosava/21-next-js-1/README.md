# Avatar app

- refresh all avatars.

## Run Locally

Clone the project

```bash
  git clone git@github.com:qaprosoft/react-laba-international-2.git
```

Go to the project directory

```bash
  cd homeworks/lizaveta-kryvanosava/21-next-js-1
```

Install dependencies

```bash
  npm install
```

Run development server

```bash
  npm run dev
```

Or build it

```bash
  npm run build
  npm start
```
